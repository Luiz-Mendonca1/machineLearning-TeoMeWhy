# Machine Learning - Téo Me Why (2025)

Este repositório contém meus estudos, experimentos e projetos práticos desenvolvidos durante o curso de Machine Learning do canal **Téo Me Why (Edição 2025)**. O objetivo é consolidar o conhecimento sobre análise de dados, modelagem preditiva e algoritmos de aprendizado de máquina.

## Estrutura do Repositório

**/notebooks**: Contém os Jupyter Notebooks desenvolvidos a partir da playlist [Machine Learning - Téo Me Why 2025](https://youtube.com/playlist?list=PLvlkVRRKOYFR6_LmNcJliicNan2TYeFO2&si=iuyYz5tnB-FZrzPf).

**/data**: Datasets e bases de dados utilizadas para treinamento e teste dos modelos.

**/src**: Scripts auxiliares e arquivos de funções modulares criados ao longo das aulas.
